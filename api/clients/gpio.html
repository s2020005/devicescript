GPIO

caution

This service is experimental and may change in the future.

Access to General Purpose Input/Output (GPIO) pins on a board. The pins are indexed 0 ... num_pins-1. The indexing does not correspond to hardware pin names, nor labels on the board (see get_pin_info command for that), and should not be exposed to the user.

  • client for GPIO service
  • inherits Sensor
import { GPIO } from "@devicescript/core"

const GPIO = new GPIO()

Commands​

configure​

Configure (including setting the value) zero or more pins. Alternative settings means the pin is controlled by other service (SPI, I2C, UART, PWM, etc.).

GPIO.configure(pin: number, mode: GPIOMode): Promise<void>

pinInfo​

Report capabilities and name of a pin.

GPIO.pinInfo(pin: number): Promise<void>

pinByLabel​

This responds with pin_info report.

GPIO.pinByLabel(label: string): Promise<void>

pinByHwPin​

This responds with pin_info report.

GPIO.pinByHwPin(hw_pin: number): Promise<void>

Registers​

reading​

For every pin set to Input* the corresponding bit in digital_values will be 1 if and only if the pin is high. For other pins, the bit is 0. This is normally streamed at low-ish speed, but it's also automatically reported whenever a digital input pin changes value (throttled to ~100Hz). The analog values can be read with the ADC service.

  • type: Register<Buffer> (packing format b)

  • track incoming values

import { GPIO } from "@devicescript/core"

const GPIO = new GPIO()
// ...
GPIO.reading.subscribe(async (value) => {
    ...
})
note

write and read will block until a server is bound to the client.

numPins​

Number of pins that can be operated through this service.

  • type: Register<number> (packing format u8)

  • read only

import { GPIO } from "@devicescript/core"

const GPIO = new GPIO()
// ...
const value = await GPIO.numPins.read()
  • track incoming values
import { GPIO } from "@devicescript/core"

const GPIO = new GPIO()
// ...
GPIO.numPins.subscribe(async (value) => {
    ...
})
note

write and read will block until a server is bound to the client.
